Privacy Policy - Konciv Support Tools

Last Updated: January 2025

Overview

Konciv Support Tools is a Chrome extension that helps users capture debugging information for support tickets and clear cache for Konciv applications. This privacy policy explains what data we access and how we use it.

Data Collection and Usage

We do NOT collect, store, or transmit any data to external servers. All data processing happens locally on your computer.

Website Content

The extension captures screenshots and page content only when you explicitly click the "Capture & Open Email" button. This data is:

  • Used to create a formatted email with debugging information
  • Downloaded to your computer as a screenshot file
  • Included in an email that YOU send manually
  • Never transmitted to our servers

User Activity

The extension captures console logs and network activity only when you explicitly request it by checking the respective checkboxes and clicking capture. This data is:

  • Included in the email body for debugging purposes
  • Never stored or transmitted to external servers
  • Only accessible when you explicitly initiate the capture

Authentication Information

The extension may access authentication tokens stored in localStorage or cookies only when clearing cache. This is necessary to preserve specific authentication keys (ns-authorization, ns-subscription-key) while clearing other data. This data:

  • Is never transmitted to external servers
  • Is only accessed locally on your device
  • Is preserved during cache clearing operations

Local Storage

The extension stores one piece of data locally on your device:

  • Email Preference: Your support email address is saved locally using Chrome's storage API so you don't have to re-enter it each time
  • This data is never transmitted to external servers
  • You can change or clear this at any time

Permissions Explained

  • activeTab: To capture screenshots and access page information (URL, title) for support tickets
  • tabs: To identify the current tab and reload pages after cache clearing
  • debugger: To capture console logs and network activity for debugging
  • scripting: To access localStorage and sessionStorage for debugging and cache clearing
  • storage: To save your email preference locally
  • browsingData: To clear cache and localStorage for Konciv websites
  • host_permissions: To work on any website where you might need to submit support tickets

Data Sharing

We do NOT:

  • Sell or transfer user data to third parties
  • Use data for purposes unrelated to the extension's functionality
  • Use data for creditworthiness or lending purposes
  • Transmit any data to external servers
  • Track or analyze user behavior

Your Control

You have complete control over:

  • What data is captured (via checkboxes)
  • When data is captured (by clicking the capture button)
  • Whether to send the email (you send it manually)
  • Your email preference (stored locally)

Contact

If you have questions about this privacy policy, please contact us at support@konciv.com.
